Question 4

Which CSS selector has the highest specificity?

Your answer:
Correct answer: C - ID selector (#id)
Explanation:

In CSS specificity hierarchy (excluding inline styles and !important), ID selectors have the highest specificity, followed by class selectors, then element selectors. The universal selector (*) has the lowest specificity.

Question 5

What is the default position value for HTML elements?

Your answer:
Correct answer: C - static
Explanation:

The default position value is "static", which means elements are positioned according to the normal document flow. Other position values (relative, absolute, fixed, sticky) remove elements from the normal flow or change their positioning behavior.
